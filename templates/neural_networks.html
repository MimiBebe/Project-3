Neural Networks

All runs shown used a sequential model with 'adam' optimization, run for 60 epochs

For the limited number of neural network variables tested, the highest accuracy was achieved when the minor factors identified by the Random Forest method (Gender, Education, Self_Employed, Married) were removed and Applicant and Co-Applicant income were treated separately (5b).

Log transformation of Total Income and/or Loan Amount had no consistent effect.

Increasing the number of network layers resulted in small accuracy decreases (compare set 3 with set 4, and 5 with 6).
